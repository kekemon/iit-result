## How to run
  - Download Xamp & Install
  - Download project
  - Start Apache & MySql
  - Open MySql admin panel
  - Create database 'iit_db'
  - Import iit_db.sql
  - Paste the project into 'xampp\htdocs\' folder

## Features
  - Login as  Teacher
  - Register Teacher
  - Add Student result
  - Generate Result
